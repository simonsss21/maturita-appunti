**Subnetting**: tecnica che permette di suddividre una rete IP in più sottoreti più piccole.
Serve a ridurre il traffico broadcast, migliorare l'organizzazione della rete, ottimizzare l'uso degli indirizzi IP.
Viene modificata la Subnet Mask, per "prendere" bit dalla parte host ed usarli per creare nuove reti. 


**VLSM**: il Variable Length Subnet Mask, è un estensione del subnetting.
La subnetting, permette di suddividere la rete in parti uguali, la VLSM invece permette di suddividerla in parti non uguali. 
Così facendo gli IP vengono usati più efficientemente.


**DNS**: Domain Name System, traduce i nomi di dominio in indirizzi IP.

**DHCP**: Dynamic Host Configuration Protocol, assegna automaticamente indirizzi IP ai dispositivi.
Assegna IP, Subnet Mask, Gateway, DNS.


**Protocollo NAT**: Network Addres Translation, traduce gli indirizzi IP privati in indirizzi IP pubblici, per permettere la navigazione.
Permette a più dispositividi usare un solo IP pubblico, dato che non può uscire dalla rete. 


**Protocollo PAT**: Port Addres Translation, è una forma di NAT; che però differenzia le connessioni tramite porte.
